## Readme

Nikomen is a free open source web based email client, developed on top of the web2py framework.


## Current email server support

Nikomen currently supports the following email services:
 * Gmail


## Running the demo

To run the demo on the current server: after cloning, go into the directory, run web2py, pick a password, it will bring up a website, submit that first password. Then just click on "nikomen".

## Future plans

There are many planned features, including adding custom mailboxes / folders / tags (depending on what your service calls that categorization), more UI integration, proper copyleft licensing, and support for more email services (including Outlook, Yahoo!, and generic mail servers)
